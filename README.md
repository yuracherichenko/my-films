# My Films 🎬

Приложение для просмотра списка любимых фильмов.

## Функции
- Просмотр списка фильмов
- Фильтрация по рейтингу
- Сортировка по рейтингу

## Технологии
- React
- JavaScript
- HTML/CSS

## Запуск
npm install
npm start
